- `Database` contains the micromagnetic simulation results, including the corresponding features including concavity, porosity, elements and component proportion，and the corresponding 
response indicators for the μ'' (such as S1_peak, S1_min, C1_peak, etc.) are used as the prediction labels for machine learning. S1 represent 2-3GHz，S2 represent 3-4GHz，C1 represent 4-5GHz. 

- `Database-Fe7Co3` contains the micromagnetic simulation results of Fe7Co3 spheres, with different concavity and porosity.
